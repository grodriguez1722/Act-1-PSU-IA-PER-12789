# Tablero Kanban

Tablero Kanban simple con Spring Boot, Spring Data JPA, H2 y frontend HTML/CSS/JS con Drag and Drop.

## Tecnologías
- Backend: Java 17, Spring Boot, Spring Web, Spring Data JPA
- Base de datos: H2 database (archivo local `data/kanban_db.mv.db`)
- Frontend: HTML5, CSS3, JavaScript Vanilla

## Ejecución
Para iniciar el proyecto puedes ejecutar el comando:
```bash
./mvnw spring-boot:run
```
Y abrir `http://localhost:8080/index.html` en el navegador.
